    Class DataSourceManager

    Manages the orchestration of data sources for research tools and Tadirah concepts.

    The DataSourceManager class is responsible for coordinating the fetching and enrichment of research tool data and Tadirah concept data from various sources. It handles the initialization of data source classes, manages error collection, and returns a unified API response object.

    Remarks

    This class fetches research tool data from a SPARQL endpoint, Tadirah concepts from a file, and enriches research tool data with additional information. It is designed to be used as a central data orchestrator in the Gatsby source plugin.

    Example

    const manager = new DataSourceManager(options, lastFetchedDate);
    const result = await manager.fetchAllData();
